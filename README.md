# React-Spotify
Music app, built with React js

## Features
- Single-Page Application
- Music Player
- Infinite Scroll
- Live Search
- Lazy Loading Images
- Easy Playlist Customization
- Easy to Create or Delete Playlists
- Skeleton Loader
- Filter Top Songs by Country
- Easy to Add Song to Playlist
- Responsive Design
